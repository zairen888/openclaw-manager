# OpenClaw Manager 🦞

OpenClaw 全功能管理器 - 桌面客户端

## 功能

| 页面 | 功能 |
|------|------|
| 📊 仪表盘 | 网关状态监控、技能/模型统计、实时日志 |
| ⚙️ 服务管理 | 启动/停止/重启 Gateway、WebSocket、AI 推理等 |
| 🧠 模型配置 | 管理 AI 模型接入 |
| 🧩 技能管理 | 启用/禁用技能模块 |
| 🔍 环境检查 | Node.js、系统、端口、内存等 8 项自检 |
| ⚡ 设置 | 端口、绑定地址、默认模型等全局配置 |
| 🔄 检查更新 | 版本管理 |
| ⛔ 关于 | 关于信息 |

## 使用方式

1. 直接下载 index.html 用浏览器打开
2. 或作为 Electron 桌面应用运行：
   `
   npm install
   npm start
   `

## 技术栈

- 纯 HTML + CSS + JavaScript
- 苹果绿主题 · 暗黑毛玻璃风格
- 无外部依赖，单文件运行
